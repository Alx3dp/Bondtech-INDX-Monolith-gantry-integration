# Cooling ducts

These are fully custom Coolign ducts I designed together with CFD analysis.
The main design goal was to make cpap cooling ducts that do not increase the tool to tool distance while providing good cooling performance.
Unlike the stock Voron gantry, on Monolith there is unused space behind that X-beam, so that is used by having the duct wrap around the X beam insead of the sides of the toolhead like with the stock ducts from Bondtech.

The current 2 options are either for a regular 2020 aluminum extrusion (or any other beam that has nothign protrude behind that) and the Carbonara CF-beam that has screws with nuts protruding behind it, for which there is a cutout in teh duct.

That *might* harm airflow a bit by creating an extra restriction but I don not believe it is too significant.

If you have any other option, like a backer behind the beam for example, contact me and I will see what I can do, contact details are in the main readme.
